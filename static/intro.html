To get started we recommend you to download PyRails and make sure you have Python installed ( We highly recommend Python 2.7 ). Pyrails is much inspired from Sails.js and Ruby on Rails. This framework intends to boost your app development and api's in Python.

This framework assumes that you know Python and web development ( basic knowledge would suffice ). PyRails is based out of Flask framework and follows MVC architecture ( not to worry if you have not heard of either ). This framework is distributed in the following architecture.

API - This folder is where you will keep your controllers and models.
config - contains all app configurations
static - will contain all your static content , be it js,css, images or html files.
templates - server side rendered html needs to be placed here.
app.py - this is the root file of the framework
We will go through on every component in details later.
