Misc supports almost all major Database one would require , we will be covering how to connect Mongo Database to our application. For other database the procedure would be same.

Prerequisite :

Make sure MongoDB is already installed on your system.
pip install pymongo
This would install the required driver for communicating with Mongo. Navigating to config/connections.py will show how how do we create a mongo object which we will later use in our models
