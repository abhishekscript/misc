Pyrails makes apps easy to write and maintain as the codebase grows. we maintain a "routes.py" that houses all our url routes , corresponding to which all functions exists We will quickly go through how to write "hello world" app.

Step 1 :

api/controllers/TestController.py It's a sample controller where you could write your logic
So basically within this controller we have defined hello that returns "Hello World". Obviously :)

Refer to the image above in case you are new to everything.

Step 2 :

The final step would be to point a URL to this function which again is easy to do.
Just go to config/routes.py and add the following code :

'GET /hello' : TestController.hello
So we are defining "/hello" as url that points to function hello which is present in TestController

Now Run the app using : python app.py
Refer to "Getting Started to learn how to run the app"

Step 3 : (We are done )

Make sure you have saved and RESTARTED your app for the new changes to take place.

So that's it , We wrote a simple function in the controller , then wrote a simple URL "/hello" and made it poin to the controller and the function. We started our app and then visited the url "localhost:5000/hello" and bingo it worked.
